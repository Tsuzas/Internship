O que esta To-Do faz?

A app permite ao utilizador anotar tarefas para não se esquecer do que tem para fazer.

  • Adicionar tarefas
  • Marcar tarefas como concluídas
  • Remover tarefas através de um seletor
  • Ver quantas tarefas estão pendentes
  • Ver quais tarefas foram concluidas

Como a app foi feita?

A app foi construída com Angular, utilizando a fragmentação de blocos de código em componentes separados. Isto permite uma melhor organização e reutilização do mesmo.

A app encontra-se dividida em duas páginas: Home e Sobre (a que se encontra agora).

Na página Home, são utilizados três componentes:

  • Input - Cria um text input e dois botões: Submit e Escolher ficheiro.
  • List - Onde as tarefas estão agrupadas. Mostra tambem as tarefas concluidas e cria três botões; "Done!", "Remove!" e "Task Quantity"
  • Item - Componente que renderiza uma tarefa individualmente.

Na página Sobre, é utilizado meramente HTML e TailwindCSS para decorar a pagina de modo a se parecer Moderno e legivel.

Os Serviços usados neste projeto servem maioritariamente como um lugar para guardar funções, de modo a que o back-end dos componentes fique mais legível e organizado.
